# Product Backlog

## Project: TaskFlow Task Management System

| ID | User Story | Priority | Acceptance Criteria | Status |
|----|------------|----------|---------------------|---------|
| PB1 | As a user, I want to create a task so that I can organize my work. | High | • Form with title, description, deadline<br>• Saves to local storage<br>• Appears in task list | Completed |
| PB2 | As a user, I want to view all tasks so that I can track my workload. | High | • List displays all tasks<br>• Shows title, description, deadline, status<br>• Loads from local storage | Completed |
| PB3 | As a user, I want to update task details so that changes can be reflected. | High | • Edit button on each task<br>• Can modify title, description, deadline<br>• Changes persist | Completed |
| PB4 | As a user, I want to delete tasks so that unnecessary tasks are removed. | Medium | • Delete button with confirmation<br>• Removes from storage and list | Completed |
| PB5 | As a user, I want to change task status so that I can track progress. | Medium | • Status buttons (To Do/In Progress/Done)<br>• Visual color indicators<br>• Updates statistics | Completed |
| PB6 | As a user, I want to set deadlines so that I can manage my time effectively. | Medium | • Date picker in form<br>• Deadline displayed on tasks<br>• Can edit deadline | Completed |

## Technical Tasks (Non-User Stories)
- Repository setup and configuration
- Basic UI/HTML structure creation
- Local storage implementation
- Final testing and bug fixes
